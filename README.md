# script-video-the-office-context-API

## Conteúdo do repositório:
- Uma aplicação em React usando Context API e React Hooks;
- Um arquivo em markdown com script para vídeo sobre gerenciamento de estados complexos com Context API e React Hooks;

### Sobre a Aplicação
Este repositório contém uma aplicação em `React` com duas páginas: uma página principal com todos os episódios da série **The Office**, renderizados em cards com uma imagem, o nome, a temporada e o número do episódio e outra que renderiza os cards dos episódios favoritados na página principal.

A aplicação utiliza a **API TvMaze** para buscar dados da série **The Office** e `Context API` com `React Hooks` para gerenciar os estados globais.

#### A aplicação permite:
- Adicionar ou remover episódios dos favoritos na página principal;
- Ir para a página de favoritos onde estarão os cards dos episódios marcados como favoritos;
- Remover episódios favoritados a partir da página de favoritos;
- Filtrar cards por temporada;
- Filtrar cards por episódio;

#### Intuito da aplicação
Esta aplicação foi desenvolvida para ser usada como exemplo em um vídeo sobre manipulação de estados usando `Context API` e para isso, há, além da aplicação em si, um arquivo em `markdown` com o passo a passo do desenvolvimento da aplicação que deve servir de base para o vídeo a ser feito.
